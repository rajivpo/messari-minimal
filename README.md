## Getting Started

```
touch .env.local
echo MESSARI_API_KEY=YOUR_API_KEY > .env.local
yarn install
yarn dev
```

Feel free to navigate to assets from the home page (top 100 assets by market cap) or any other asset by going to /asset/$assetOfInterest
